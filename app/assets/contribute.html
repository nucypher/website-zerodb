ZeroDB Hadoop Cloud Community

A New Open Source Community for Hadoop

Hadoop reached 1.0 in 2011. In the five years since, over 80% of deployments in the enterprise have been on-premise. This is because the native security controls in Hadoop are not sufficiently robust to meet very demanding enterprise security requirements.
The ZeroDB Hadoop Cloud Community will deliver a security layer for the Hadoop ecosystem that guarantees on-premise, enterprise-grade security in the public cloud.
